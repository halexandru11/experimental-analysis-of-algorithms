# Experimental Analysis of Algorithms

University homework repository for the course **Experimental Analysis of Algorithms (EAA)** at the **Faculty of Computer Science, UAIC**.

## Course Details
- **Subject:** Experimental Analysis of Algorithms
- **Course Website:** [UAIC - EAA](https://edu.info.uaic.ro/analiza-experimentala-a-algoritmilor/)

## Implementation
- **Tools:** [Google OR-Tools](https://developers.google.com/optimization)
- **Algorithms:** Genetic Algorithms & Constraint Programming

## Final Project
**Water Distribution Network Problem**  
Solving network optimization using OR-Tools and Genetic Algorithms.
